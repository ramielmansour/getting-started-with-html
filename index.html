Lesson 1 Notes - HTML Basics

HTML
HTML attributes consist of a name = a "value".
Anchor Tag = < a >
<a href="website">Description <a> equates to <tag attribute="value">Content </tag>

Image Tags
Image Tags are written as <img src="url" alt"text">

Image tags are considered Void Tags, or tags that do not contain a closing tag.
Closing tags are not needed because there is no content associated with void tags.

White Space
Spaces or line breaks in text alone do not translate in HTML.

<br> represents a link break, it is an inline type of line break (no "invisible borders")
<br> is also a void tag, therefore has no content.

<p> is a paragraph tag, has content, and requires a closing tag.
Has an invisible box that has height and width.

Examples of Inline Elements

    <b>Bold</b>
    <em>Italics</em>
    <img src> for Images
    <span></span>
    <a> </a>

Examples of Block Elements
